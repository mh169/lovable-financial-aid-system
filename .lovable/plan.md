

# خطة إصلاح وتحسين نظام الدعم المالي وتجهيزه للتنزيل

## ملخص المشروع
النظام الأصلي هو تطبيق Express + SQLite + React يعمل على شبكة محلية بدون إنترنت. سأقوم بإصلاح وتحسين المشروع الأصلي ثم تجهيزه كملف قابل للتنزيل والتشغيل مباشرة.

---

## التعديلات المطلوبة

### 1. إصلاح أعمدة جداول طباعة المسيرات

**مسير الصرف** — الأعمدة الحالية ناقصة. يجب أن تكون:
رقم الاستمارة، الرقم الطبي، اسم المريض، نوع الطلب المرفوع/جهة الرفع، تفاصيل الاحتياج، المبلغ ريال يمني، المبلغ دولار، جهة التسليم، اسم المستلم، ملاحظة للمستلم، رقم هاتف المستلم، صلة القرابة/الصفة، المرفقات، ملاحظات

**مسير الإخلاء** — الأعمدة:
رقم الاستمارة، الرقم الطبي، اسم المريض، نوع الطلب المرفوع/جهة الرفع، تفاصيل الاحتياج، المبلغ ريال يمني، المبلغ دولار، جهة التسليم، اسم المستلم، صلة القرابة/الصفة، المرفقات، ملاحظات

**مسير التحصيل** — الأعمدة:
رقم الاستمارة، الرقم الطبي، اسم المريض، نوع الطلب المرفوع/جهة الرفع، تفاصيل الاحتياج، المبلغ ريال يمني، المبلغ دولار، جهة التسليم، اسم المستلم، ملاحظة للمستلم، رقم هاتف المستلم، صلة القرابة/الصفة، ملاحظات

### 2. تحسين الطباعة للمسيرات الثلاثة
- احتواء جميع الأعمدة في عرض الصفحة A4
- ترقيم الصفحات
- تكرار الترويسة (header) والتوقيعات في كل صفحة عند الامتداد لأكثر من ورقة

### 3. إصلاح رقم الاستمارة في نماذج الطباعة
- في استمارة الصرف الداخلية والخارجية: تغيير "رقم استمارة المساعدة" ليعرض `formNumber` (رقم الاستمارة)

### 4. إضافة تبويب الصلاحيات (نظام المستخدمين)
- إضافة جدول `users` في SQLite (اسم المستخدم، كلمة المرور المشفرة، الدور)
- الأدوار: مدير النظام، مستخدم عادي، مستخدم مشاهد
- صفحة تسجيل الدخول
- تبويب إدارة الصلاحيات في الشريط الجانبي (للمدير فقط)
- 3-5 مستخدمين افتراضيين
- حماية العمليات حسب الدور (المشاهد يرى فقط، العادي يضيف ويعدل، المدير له كل الصلاحيات)

### 5. التحقق من المشاريع
- اختبار وإصلاح صفحة المشاريع (إضافة مشروع بإدخال اسم مشروع، من بند، مبلغ البند)

## التنفيذ التقني

سيتم العمل على المشروع الأصلي المستخرج (`/tmp/extracted/`) وتعديل الملفات التالية:

| الملف | التعديل |
|-------|---------|
| `server/init-db.ts` | إضافة جدول users مع بيانات افتراضية |
| `server/routes.ts` | إضافة مسارات المصادقة وإدارة المستخدمين |
| `server/storage.ts` | إضافة دوال إدارة المستخدمين |
| `shared/schema.ts` | إضافة أنواع User والتحقق |
| `client/src/pages/Vouchers.tsx` | إصلاح أعمدة طباعة مسير الصرف + ترقيم صفحات + تكرار ترويسة |
| `client/src/pages/Evacuations.tsx` | إصلاح أعمدة طباعة مسير الإخلاء + ترقيم + ترويسة |
| `client/src/pages/Collections.tsx` | إصلاح أعمدة طباعة مسير التحصيل + ترقيم + ترويسة |
| `client/src/lib/printTemplates.ts` | إصلاح "رقم استمارة المساعدة" → رقم الاستمارة |
| `client/src/pages/Login.tsx` | صفحة تسجيل دخول جديدة |
| `client/src/pages/Permissions.tsx` | صفحة إدارة الصلاحيات جديدة |
| `client/src/App.tsx` | إضافة مسارات Login و Permissions + حماية المسارات |
| `client/src/components/layout/Sidebar.tsx` | إضافة تبويب الصلاحيات + عرض اسم المستخدم الحالي |

## التسليم
- أرشيف `.tar.gz` جاهز للتنزيل يحتوي على المشروع الكامل
- يعمل بأمر `start.sh` أو `start.bat`
- لا يحتاج اتصال بالإنترنت بعد تثبيت الحزم أول مرة
